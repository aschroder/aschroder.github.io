In a bid to help solve a weird 404 error that Kevin has been experiencing with the self test I offered him a pre-release copy of my upcoming ‘unified’ Magento email extension. It will support both SMTP and Google Apps/Gmail along with a few new features.

Anyway, if anyone is keen to try the extension please contact me or flick me an email or post a comment and let me know, I’ll send through the files, I’d rather not post it for public availability here yet, because it’s not as stable as I’d like. But expect it very soon!

If there is enough interest in this little beta testing program, I’ll scrounge up some free magento hosting from one of the magento hosts I have reviewed and make it a little prize draw for the beta testers. So how about that? Not only do you get to be one of the first try try the extension, you might get some free magento hosting to go with it! Lucky!

Note: It is not fully tested yet, and so not ready for production, but I thought I’d offer/ask if anyone is interested in helping out with testing (and possibly translation). I can send through the pre-release extension and you could give me some of your first impressions on it, try it out with whatever weird and wonderful email server you connect to and generally try to break it on various versions of Magento.
