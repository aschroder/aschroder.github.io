1) Use jQuery ‘in view’ event plugin to detect when the footer is coming in to view.

jQuery('#footer-outer')
    .bind('inview', function (event, visible) {

2) When it’s in view – scroll up the menu block.

jQuery('.ribbon')
    .animate({ top: -400 });

3) When it’s back out of view scroll it back down.

jQuery('.ribbon')
    .animate({ top: 0 });

4) Extra merit – I added a little down arrow &darr; that appears when the menu is scrolled away, that will manually pull it down.


↓

and

function showMenu() {
    jQuery('.ribbon')
    .animate({ top: 0 });
    jQuery('.down-arrow')
    .animate({ top: -30 });
}

And there you have it.
